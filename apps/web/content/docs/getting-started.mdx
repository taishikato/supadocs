# Getting Started

Supadocs は Next.js と Supabase をベースにした AI 対応ドキュメントテンプレートです。

## セットアップ

1. リポジトリをクローンします。
2. `pnpm install` を実行します。
3. `supabase start` でローカル Supabase を起動します。

## プロンプト検索

`pnpm dev` でローカルサーバーを起動し、チャットから質問すると関連するドキュメントが引用付きで返されます。
