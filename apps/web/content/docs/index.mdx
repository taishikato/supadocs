title: "Supadocs Docs Introduction"
description: "Meet the moving parts behind Supadocs and how the docs experience fits together."
---

Supadocs is here to keep documentation updates fast, auditable, and a little less stressful for everyone involved.

- Track every change with auditing that follows collection, drafting, and review runs.
- Keep knowledge bases fresh by syncing Markdown and MDX in near real time.
- Collaborate confidently with Supabase-backed storage and permission controls.
